# Architecture diagrams

The diagrams in this directory are written in [D2](https://d2lang.com), so
they can be reviewed and maintained as code.

To regenerate the rendered images after editing a `.d2` file, run from the
repository root (requires the [d2 CLI](https://d2lang.com/tour/install)):

```bash
d2 diagrams/cc_tee_container.d2 images/CC_TEE_container.svg
d2 diagrams/coco_v1_kata.d2 images/COCO_ccv1_TEE.svg
d2 diagrams/coco_v1_peerpods.d2 images/COCO_ccv1_peerpods_TEE.svg
```
